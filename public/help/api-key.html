Google Cloud API キー取得方法（Translation）

このアプリは Cloud Translation API を使用して言語判定と翻訳を行います。

前提

  • Google アカウント
  • 組織の方針により、課金設定が必要になる場合があります

1. プロジェクト作成

  1. Google Cloud Console を開く
  2. プロジェクト セレクタから「新しいプロジェクト」を作成

2. Translation API を有効化

  1. 対象プロジェクトを選択
  2. 「API とサービス」→「ライブラリ」
  3. 「Cloud Translation API」を検索して「有効にする」

3. API キー作成

  1. 「API とサービス」→「認証情報」
  2. 「認証情報を作成」→「API キー」
  3. 発行されたキー（例: AIza...）を控える

4.（推奨）キーを制限

企業利用では、可能な範囲でキーを制限してください。

  • API の制限: Cloud Translation API のみに限定
  • アプリケーションの制限: 可能なら HTTP リファラで http://127.0.0.1:5173/* を許可

5. アプリへ設定

  1. アプリ画面の「API」タブを開く
  2. API キーを貼り付けて「保存」
  3. 「接続テスト」が成功したら準備完了
